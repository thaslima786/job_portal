# 💼 Job Portal Web Application

## 📌 Project Overview

The Job Portal Web Application is a web-based platform developed using PHP and MySQL that allows users to register, log in, and access a personalized dashboard. The project demonstrates user authentication, database connectivity, session management, and responsive web design.
### 👤 User Authentication

* User Registration
* User Login
* Secure Password Hashing
* Session Management
* User Logout
## 🛠 Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* XAMPP
* phpMyAdmin
